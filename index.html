TikTok API Integration Demo

Login with TikTok, simulate draft upload, try Web Share, and export profile + video stats as JSON. This is a non-production demo for review.

Data export: profile + video stats exported as JSON (demo). No real posting happens in this environment.

Privacy Policy Terms of Service

1) Login

Opens TikTok OAuth (Login Kit) with scopes user.info.basic, video.upload, video.publish. Redirect to callback.html.

2) Draft upload (demo)

Simulates posting a local video as a draft via Content Posting API. In the demo, we show a success toast.

3) Share

Uses Web Share API (or clipboard fallback) to share the site URL — demo of Share Kit UX.

4) Export

Creates a JSON with mock profile and video stats to demonstrate Data Portability flow.
